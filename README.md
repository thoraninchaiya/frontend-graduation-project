# SALAFEX End project

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3300
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```


### Status
```
[u] = update
[c] = change
[n] = new
[e] = edit
```
